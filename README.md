# Instagram Account Creator Bot

Professional Telegram bot for automated Instagram account creation with advanced features.

## 🌟 Features

- **Bulk Creation**: Create 1-50 Instagram accounts at once
- **12 Email Providers**: Tier 1 premium + temporary email services
- **Flexible Configuration**: 
  - Username: Random, Custom Prefix, or Custom List
  - Full Name: Random, Same, List, or Empty
  - Password: Custom (same for all) or Random (unique each)
- **Smart Captcha Solving**: Free methods + 2Captcha integration
- **Real-time Progress**: Live updates during account creation
- **Anti-Suspension**: Human-like behavior, delays, profile completion
- **Secure Storage**: Encrypted credential storage

## 📋 Requirements

- Python 3.11+
- Chrome/Chromium browser
- Telegram Bot Token
- 2Captcha API Key (optional)
